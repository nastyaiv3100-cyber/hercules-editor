🔗 Как опубликовать игру
Игра публикуется бесплатно через GitHub Pages. Это занимает 3 минуты.
1
Скачайте файл игры — кнопка ниже
2
Создайте бесплатный аккаунт на github.com
Затем: New repository → назовите my-games → Create repository
3
Загрузите файл: Add file → Upload files → перетащите скачанный .html → Commit changes
4
Включите публикацию: Settings → Pages → Source: Deploy from branch → main → Save
Через 1–2 минуты появится ваша ссылка:
Введите ваш GitHub-логин:
https://ВАШ_ЛОГИН.github.io/my-games/hercules-game.html
5
В Genially: Others → Insert external content → вставьте ссылку → Insert → растяните блок на весь слайд ✅
